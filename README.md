# BI & Data Engineering Assignments

This repository contains deliverables for three Business Intelligence and Data Engineering assignments focused on ETL development, SQL Server Data Warehousing, SSIS, and Power BI dashboard design.

## Assignments

### 1. Monthly Summary Dashboard

Design and build a Monthly Summary Dashboard using CSV data, an ETL pipeline, SQL Server, and a Star Schema model.

Key work includes:
- Cleaning and loading CSV data
- Building Fact and Dimension tables
- Creating a `Dim_Date` table for time intelligence
- Developing SSIS packages for ETL
- Designing an interactive Power BI dashboard

### 2. Sigma Capital Market Overview Analysis

Analyze Sigma Capital Market Overview and QSE Trading Reports to identify data points useful for reporting and dashboarding.

Key areas include:
- Investor participation
- Trading activity by investor type
- Market breadth metrics
- Sector activity
- Block trades
- KPI and dashboard recommendations

### 3. EFG One App Analysis & Dashboard Design

Analyze the EFG One mobile app from a BI perspective and design executive dashboards for customer behavior, digital adoption, trading activity, and business performance.

Key areas include:
- Customer analytics
- Login and usage analytics
- Trading activity
- Portfolio performance
- Engagement and feature adoption
- Executive dashboard design

## Tools & Technologies

## Goal

The goal of these assignments is to practice end-to-end BI development, from data preparation and warehouse design to KPI definition and dashboard creation.
